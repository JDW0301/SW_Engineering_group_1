# Agent Governance Documents

이 디렉터리는 이 프로젝트의 AGENTS 관련 문서를 관리하기 위한 공간이다.

## 문서 역할

- 프로젝트 루트의 `AGENTS.md`: 규범(normative) 문서
- 프로젝트 루트의 `AGENTS-summary.ko.md`: 사람 검토용 비규범(non-normative) 요약
- `archive/`: 과거 AGENTS 관련 문서 보관

## 운영 원칙

- 에이전트는 기본적으로 프로젝트 루트의 `AGENTS.md`만 읽는다.
- 이 프로젝트 작업에서는 홈 디렉터리의 전역 AGENTS 규칙보다 프로젝트 루트의 `AGENTS.md`가 우선한다.
- `AGENTS-summary.ko.md`는 AGENTS 수정/검토 또는 사용자 요청 시에만 읽는다.
- 과거 AGENTS 문서는 `archive/`에서만 관리한다.
