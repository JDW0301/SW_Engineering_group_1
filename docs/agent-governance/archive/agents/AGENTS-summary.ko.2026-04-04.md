# AGENTS 한국어 요약

> 이 문서는 사람 검토용 비규범 요약이다.
> 규범 문서는 프로젝트 루트의 `AGENTS.md`이다.

## 핵심 원칙

- 기본적으로 최신 active 문서만 읽는다.
- 먼저 인덱스/README를 보고 필요한 세부 문서만 연다.
- 과거 문서는 예외 조건에서만 읽는다.
- 작업은 도메인/컨텍스트 단위로 분류한다.
- 구현 전에 규칙, 정책, 상태, 이벤트를 먼저 파악한다.

## 기본 읽기 순서

1. `docs/ddd/README.md`
2. `docs/ddd/active/*`
3. `docs/specifications/README.md`
4. `docs/specifications/active/*`
5. `immediate-mvp-notes.md`는 현재 결정사항이 필요할 때만

## 기본적으로 읽지 않는 문서

- `docs/project-handbook/*`
- `docs/specifications/archive/*`
- `docs/ddd/archive/*`

## 프로젝트 기본 컨텍스트

- Case Management
- Customer Portal
- Commerce Integration
