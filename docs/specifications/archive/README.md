# Archive 문서 안내

이 디렉터리는 과거 문서군 또는 비교 기준 문서를 참조하기 위한 안내 공간이다.

## 현재 보관 기준

- 기존 1차 문서 세트: `../project-handbook/`

## 비교 원칙

- `project-handbook`는 문제 정의, 요구사항 초안, 기능 방향을 정리한 **기준 입력 문서군**으로 본다.
- `specifications/active`는 구현 지향으로 재정리된 **현재 기준 상세 명세 문서군**으로 본다.
- 새 버전이 생기더라도 `active`만 현재 기준으로 유지하고, 이전 기준은 `archive`에서 링크 또는 요약으로 추적한다.
- 과거 문서는 기본 읽기 대상이 아니며, `document-version-policy.md`에 정의된 예외 조건에서만 선택적으로 참조한다.
